# Scientific Context Notes

A browser extension (Chrome Manifest V3) providing a research workflow layer directly in the browser. It combines contextual annotations on web pages and PDFs, project-based organisation, and automatic citations/bibliography generation using CSL styles.

## Purpose

Scientific Context Notes helps researchers, analysts, activists, and students move from reading sources in the browser to writing well-documented reports and articles, without fragmented notes across multiple tools.

## Key Goals

- Provide project-centric organisation of online sources and notes.
- Anchor notes to precise parts of web pages, PDFs, tables, and charts.
- Automatically generate citations and bibliographies following configurable citation styles.
- Support collaborative research workflows while keeping data local-first and privacy-aware.
